# Market Analysis Package

## 概述

本包包含了市场杠杆分析系统的完整数据集、算法说明和可视化设计文档。所有文件都基于真实的金融市场数据，为市场风险分析和投资决策提供科学依据。

## 核心指标摘要

| 指标名称 | 英文名称 | 计算公式 | 主要用途 |
|---------|---------|---------|---------|
| 市场杠杆比率 | Market Leverage Ratio | Margin Debt / Market Cap | 衡量整体市场杠杆水平 |
| 货币供应比率 | Money Supply Ratio | Margin Debt / M2 Money Supply | 评估杠杆占用的社会资源 |
| 利息成本分析 | Interest Cost Analysis | Margin Debt × Fed Rate | 分析金融负担 |
| 杠杆变化率 | Leverage Change Rate | (本期-上期)/上期 × 100% | 监控杠杆动态 |
| 投资者净资产 | Investor Net Worth | Market Cap - Margin Debt | 评估净杠杆位置 |
| 脆弱性指数 | Vulnerability Index | Leverage Z-Score - VIX Z-Score | 主要风险指标 |
| VIX vs 杠杆 | VIX vs Leverage | 相关性分析 | 波动率比较分析 |
